postion 속성
- static : 순차적인... html에서 태그를 작성 => 작성한 순서대로 화면 UI  출력!!!

- relative : 상대적인... static의 기본 포지션 좌표값을 기준으로 상대적인 좌표값을 새롭게 정의하고 싶을 때

- absolute : 절대적인...기존 static의 포지션 좌표값과는 완전 무관하게 새롭게 포지션 좌표값을 부여할 수 있는 속성 

- fixed : 웹 페이지 안에서 특정 좌표에 포지션을 고정!!!
- sticky : 스티커 처럼 특정 좌표에 도착하면 고정!!!
